Basic Tags

Block Level Text Containers

The <p> creates a paragraph

The <blockquote> specifies a section that is quoted from another source.
The <pre> defines preformatted text.

Inline Text Styling

The <em> is used to define emphasized text. Usually in italics.
The <i> is used to define a part of text in an alternate voice or mood. Usually in italics.
The <strong> is used to define text with strong importance. Usually in bold.
The <b> specifies bold text without any extra importance.
The <code> is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.
The <a> tag defines a hyerlink. The <img> is used to embed an image in an HTML page.

The <h1> header should represent the main heading/subject for the whole page.

The <h2> header is used after the <h1> header and is usually of less importance than the <h1> header.

The <h3> header is used after the <h2> header and is usually of less importance than the <h2> header.
