HTML colors are specified with predefined color names. This text is centered using the <text-align> tag. The <text-transform> tag is used to control the capitalization of text. The <line-height> tag is used to change the distance between lines. The <letter-spacing> tag is used to set the horizonal spacing between characters. The <font-family> specifies the font used. The <font-size> tag specifies the size of the text. The <font-style> tag specifies the font style for text. The <background-color> tag sets the background for the specified section (in this case the body). The <background-image> tag sets the background to an image of choice.

  • The <list-style-type> tag sets the marker of a list item element

Using the <width> tag and <height> tag I am able to manipulate the width and height of the inserted image. The <border> tag creates a border around a specified area (can manipulate the border design, color, size, etc.). The <margin> tag creates space around elements, outside of any defined borders. The <padding> tag creates space around an element's content, inside of any defined borders.
