The tags used in the snippet above are the <p> and <a>. You can find information about the <p> here. You can also find information about the <a> here. In this example the <p> is the parent of the <a> because it is within the <p>. This means that the <a> is the child of the <p>. <p> is the start of a paragraph and </p> closes the paragraph. <a> defines a hyperlink and </a> closes the hyperlink tag. The attribue names of the <a> are href, id, class, and target. The href attribute has the value "https://google.com", the id attribute has the value "the-best-link", the class attribute has the value "banana", and the target attribute has the value "_blank". The href attribute with value "https://google.com" identifies the URL of the page that the link will go to. The id attribute with value "the-best-link" specifies a unique name to easily maipulate the element. The attribute class with value "banana" is used to specify a class named banana to easily manipulate the element. The target attribute with value "_blank" opens the document in a new link or tab.
