Basic Tags

Block Level Text Containers

The <p> tag creates a paragraph

The <blockquote> tag specifies a section that is quoted from another source.
The <pre> tag defines preformatted text.

Inline Text Styling

The <em> tag is used to define emphasized text. Usually in italics.
The <i> tag is used to define a part of text in an alternate voice or mood. Usually in italics.
The <strong> tag is used to define text with strong importance. Usually in bold.
The <b> tag specifies bold text without any extra importance.
The <code> tag is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.
The <a> tag defines a hyperlink. The <img> tag is used to embed an image in an HTML page.

The <h1> tag header should represent the main heading/subject for the whole page.

The <h2> tag header is used after the <h1> tag header and is usually of less importance than the <h1> tag header.

The <h3> tag header is used after the <h2> tag header and is usually of less importance than the <h2> tag header.

The <table> tag defines an HTML table. The <tr> tag defines a table row. The <td> defines each table data/cell (elements are regular and left-aligned). The <th> defines a table header (the elements are bold and centered).
